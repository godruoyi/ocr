## Feature

- 自定义缓存支持；
- 符合 PSR 标准，可以很方便的与你的框架结合；
- 命名不那么乱七八糟；
- 支持目前市面多家服务商
- 查看更新日志

## Support

- 百度 OCR
- 腾讯 OCR
- 阿里 OCR

## Requirement

- PHP >= 7.0
- composer

## Installation

> 注意此版本**不兼容** 1.x 版本，若你在寻在 1.x 版本，请查看 [这里](https://github.com/godruoyi/ocr/tree/1.1.4)。

```
composer require "godruoyi/ocr:^2.0"
```
